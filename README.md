# VMware CF3 - West Wild

## Description
This repository contains the VMware West Wild virtual machine for the CF3 Cyber Security lab.

## Download the Virtual Machine

The VM files are hosted on Google Drive because they exceed GitHub's file size limit.

**Google Drive Link:**

https://drive.google.com/drive/folders/193llebfHNa5YdNUnh8DgSSfqMSEkeYDl?usp=sharing

## Files Included

- WEST_WILD.ovf
- WEST_WILD.mf
- WEST_WILD-disk1.vmdk

## How to Import

1. Download all the files from Google Drive.
2. Keep all the files in the same folder.
3. Open VMware Workstation.
4. Click **File → Open**.
5. Select **WEST_WILD.ovf**.
6. VMware will automatically import the virtual machine.
